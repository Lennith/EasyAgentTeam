# MiniMax Vision Skill
This skill provides access to MiniMax's image understanding capabilities via MCP.
## When to Use This Skill
- User asks to "describe this image", "analyze this image", "what's in this image"
- User uploads an image and wants analysis
- User asks to "look at" or "understand" an image
- Image recognition, OCR, screenshot analysis tasks
## MCP Tools Available
This skill loads the `minimax-coding-plan` MCP server which provides:
- `understand_image` - Analyze and describe images
- Image understanding and vision capabilities
## Usage
When this skill is loaded, the agent can use the MCP tools to analyze images.
