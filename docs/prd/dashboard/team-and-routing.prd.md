# 团队与路由 PRD（最后更新：2026-04-16）

## 状态

- 文档状态：`实装`

## 目标

团队与路由页面负责编辑 team 成员、消息路由、任务路由和 discuss 轮次策略，为 project/workflow 提供可复用配置。

## 当前有效能力

- team 列表与详情
- team 成员选择
- route table、task assign route table、discuss rounds 编辑
- agent model configs 配置与回显

## 非目标

- 不在前端做路由授权判断
- 不在前端定义 provider/model 合法性规则
