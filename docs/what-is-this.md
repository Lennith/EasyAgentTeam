# What Is This

EasyAgentTeam is a task-driven multi-agent orchestration framework for local software delivery workspaces.

It focuses on two outcomes:

- Orchestration: route tasks across role agents with dependency-aware execution.
- Observability: inspect task tree, timeline, sessions, and runtime events for each run.

Execution modes:

- Project mode: task-first collaboration in a concrete project workspace.
- Workflow mode: phase-level orchestration where agents decompose and report progress.

Typical use cases:

- Team task decomposition with explicit owner/dependency control.
- Repeatable workflow runs with templates and role routing.
- Regression verification with scenario-based E2E baselines.
